<h1 align="center">💬 QuickChat</h1>

<p align="center">
  A real-time chat application built using the MERN stack with Socket.IO
</p>

<hr/>

<h2>🚀 Features</h2>
<ul>
  <li>🔴 Real-time messaging using Socket.IO</li>
  <li>🔐 User authentication (Login / Signup)</li>
  <li>👤 User profiles</li>
  <li>💬 One-to-one chat system</li>
  <li>📱 Responsive UI (works on mobile & desktop)</li>
  <li>⚡ Fast and smooth experience</li>
</ul>

<hr/>

<h2>🛠️ Tech Stack</h2>
<ul>
  <li><b>Frontend:</b> React, Tailwind CSS</li>
  <li><b>Backend:</b> Node.js, Express.js</li>
  <li><b>Database:</b> MongoDB</li>
  <li><b>Real-time:</b> Socket.IO</li>
</ul>

<hr/>

QuickChat/

<h2>⚙️ Installation</h2>

<h3>1. Clone the repository</h3>
<pre>
git clone https://github.com/your-username/QuickChat.git
cd QuickChat
</pre>

<hr/>

<h2>▶️ Run the App</h2>

<b>Start Backend:</b>
<pre>
cd server
npm run dev
</pre>

<b>Start Frontend:</b>
<pre>
cd client
npm run dev
</pre>
